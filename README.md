# Patient-Management-System-Fast-API-
Developed a scalable Patient Management System using FastAPI, supporting CRUD operations, data validation with Pydantic, and real-time BMI and Verdict computation.
## Features
- Add patient
- View patient
- Update patient
- Delete patient
- BMI calculation
- Verdict Calculation

## Tech Stack
- FastAPI
- Python
- Pydantic

## Run Project
python -m uvicorn Patient_Management_System:app --reload
